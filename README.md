## Building

```shell
$ git clone https://github.com/raceup/yentl.git
$ cd yentl
$ cd script
$ chmod +x bootstrap
$ ./bootsrap
$ bash run.sh
```
then open a browser [here](http://127.0.0.1:4000/)
